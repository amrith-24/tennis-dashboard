# tennis-dashboard

This project is an interactive tennis analytics dashboard built using Python and Streamlit.
Features
View wins and losses for selected players
Explore match data interactively
Designed to grow into a full sports analytics and machine learning project
Technologies Used
Python
pandas
Streamlit
GitHub
Future Improvements
Surface-specific statistics
ATP data integration
Match outcome prediction using machine learning
Interactive visualizations
How to Run
pip install -r requirements.txt
streamlit run app.py
